## Installation:

Via Composer
``` bash
$ composer global require free-donut/page-loader
```

## Usage:

``` bash
$ page-loader (-h|--help)
```
``` bash
$ page-loader <url> [-o <dir>]
```
``` bash
$ page-loader (-v|--version)
```
Options:

  -h --help                 display help for command

  -v --version              output the version numer

  -o --output <dir>         output dir [default: ./]
